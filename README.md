# flight-travel-management
Complete full stack project with Redis, Spring Security, Nginx and Kafka
Full stack development flight travel management using Angular + Spring boot + MSQL + Redis + Spring Security, Nginx and Kafka. 
It is complete flight management system to track the flight in the event of cancellations. It helps rebook the flights on distructions, inform hotel, cafe and 
other parties linked. 

It makes the user experience easy.
